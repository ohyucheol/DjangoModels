# D01. 간행물 중 도서

## 설계의 기준

### 기본 시나리오
1. 온라인 서점을 운영하는 甲은 이용자에게 도서를 판매하고자 한다.
2. 도서관을 운영하는 乙은 도서를 수집·정리·보존하고 이용자에게 대여·열람 서비스를 제공하고자 한다.
3. 출판사를 운영하는 丙은 출판한 도서의 목록을 관리하고자 한다.
4. 유통사를 운영하는 丁은 보유한 도서의 재고를 관리하고자 한다.
